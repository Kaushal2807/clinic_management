⚠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Blank page or errors?
Solution: 
  1. Make sure Apache and MySQL are running in XAMPP
  2. Check XAMPP Control Panel - both should be green
  3. Restart both services if needed

Problem: "Connection failed"?
Solution:
  1. Start MySQL in XAMPP Control Panel
  2. Verify MySQL is running (green status)
  3. Try the browser setup again

Problem: Can't access any page?
Solution:
  1. Verify URL: http://localhost/clinic_management/
  2. Check folder path: /opt/lampp/htdocs/clinic_management/
  3. Make sure Apache is running

Problem: Import errors in phpMyAdmin?
Solution:
  1. Use the browser-based setup instead (easier)
  2. Go to: http://localhost/clinic_management/setup.php
